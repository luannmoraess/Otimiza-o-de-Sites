Comandos Git

clonando


Inicialização das atividades

  • Git fetch --all

    sincroniza o repositorio local com o repositorio remoto

    Git pull

    fax o dowload das atualizações do servidor remoto


Finalizações das Atividades

  • Git Status

    lista as modificações

    Git add

    adiciona as modificações no servidor local

    Git commit -m "[texto]"

    adiciona uma commit no github com o texto que voce quiser

    Git push

    faz o upload dos novos aquivos para o servidor remoto
